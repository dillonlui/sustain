# Sustain Brand Assets

This folder contains the recreated final concept assets for the Sustain brand.

## What changed in this recreation
This pass re-centers the assets on the actual decisions from the conversation:

- **Sustain** is the chosen name.
- The mark is a **single flowing wave**.
- The wave is rendered as **one stroke only**.
- The ripple field is **subtle and secondary**.
- The overall feel is **calm, premium, organic, and Apple-like**.
- No inner border / inner highlight / ribbon-within-ribbon effect.

## Included files
- `sustain-icon-dark.svg` — primary dark app icon
- `sustain-icon-light.svg` — light app icon variant
- `sustain-icon-monochrome-dark.svg` — simplified monochrome dark variant
- `sustain-symbol-dark.svg` — compact symbol-only mark
- `sustain-mark-only.svg` — horizontal mark without wordmark
- `sustain-wordmark-dark.svg` — dark wordmark lockup
- `sustain-wordmark-light.svg` — light wordmark lockup
- `sustain-brand-brief.md` — detailed design brief
- `brand-data.json` — structured palette / direction data
- `preview.html` — quick browser preview
