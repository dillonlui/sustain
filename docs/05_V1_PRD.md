# WorshipPad V1 Product Requirements

## Objective

Enable worship leaders to run pads, click tracks, countoffs, and transitions during live worship services.

---

## Success Criteria

A worship leader can:

1. Build a setlist.
    
2. Assign keys and BPMs.
    
3. Configure outputs.
    
4. Run a complete worship service.
    
5. Transition between songs.
    
6. Recover from common mistakes.
    

Without needing additional software.

---

## Core Features

### Song Library

Create reusable songs.

Fields:

- Title
    
- Default Key
    
- Default BPM
    
- Time Signature
    
- Pad Pack
    

---

### Setlist Builder

Capabilities:

- Add songs
    
- Reorder songs
    
- Override key
    
- Override BPM
    

---

### Live Service Screen

Displays:

- Playing Song
    
- Cued Song
    
- Key
    
- BPM
    
- Pad Status
    
- Click Status
    

Primary Controls:

- Start Song
    
- Next Song
    
- Previous Song
    
- Stop
    

Secondary Controls:

- Start Click
    
- Stop Click
    
- Start Pad
    
- Stop Pad
    

---

### Audio Setup

Configure:

- Pad Output
    
- Click Output
    

Persist settings locally.

---

### System Check

Validate:

- Audio outputs
    
- Audio devices
    
- Pad assets
    

Block playback if critical requirements are missing.

---

## Non-Functional Requirements

### Reliability

The app should never silently fail.

### Offline

The app must function without internet access.

### Performance

Song transitions should feel immediate.

### Simplicity

A volunteer should understand the Live Service screen immediately.
