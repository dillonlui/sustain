# WorshipPad Anti-Requirements

## WorshipPad Is Not

- A DAW
    
- An Ableton replacement
    
- A recording application
    
- A music production tool
    
- A mixer
    
- A worship planning platform
    
- A lyrics application
    
- A chart management application
    

---

## Explicitly Out of Scope

### Audio Production

- Recording
    
- Audio editing
    
- Waveform editing
    
- Time stretching
    
- Pitch correction
    
- Audio effects
    
- Plugin hosting
    
- VST support
    
- AU plugin support
    

### Music Creation

- MIDI sequencing
    
- Piano roll
    
- Loop editing
    
- Sample editing
    
- Pad generation
    
- Synth engines
    

### Live Production

- DMX
    
- Lighting control
    
- ProPresenter integration
    
- Video triggering
    

### Collaboration

- Accounts
    
- Cloud sync
    
- Team sharing
    
- Multi-user editing
    

### Worship Planning

- Scheduling
    
- Volunteer management
    
- Song planning
    
- Planning Center integration
    

### Arrangement Features

- Verse markers
    
- Chorus markers
    
- Bridge markers
    
- Sections
    
- Timelines
    
- Scene views
    

---

## V1 Rule

Any feature request must answer:

"Does this directly help a worship leader run pads, click, countoffs, or transitions on Sunday morning?"

If not, reject it.
